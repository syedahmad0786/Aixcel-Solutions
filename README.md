# Aixcel Solutions

Production source for [aixcelsolutions.com](https://aixcelsolutions.com).

The site is a self-contained static HTML company website. It presents Aixcel's services, documented system work, delivery process, founder profile, FAQs, and Manhaj as the company's core product.

The primary booking CTA is the same Cal.com event used by AhmadBukhari.com:

`https://cal.com/ahmad-bukhari/ai-consultancy-call-with-ab`

The founder photograph is served from AhmadBukhari.com so the company and portfolio sites use the same maintained asset. No build step or environment variables are required.

Vercel deploys the production site from the `main` branch.
